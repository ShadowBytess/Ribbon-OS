# 🎀 Ribbon OS

**Soft design. Powerful core.**

Ribbon OS is a custom Linux distribution based on Linux Mint, designed to provide a clean, aesthetic, and creator-focused experience out of the box.

---

## ✨ Overview

Ribbon OS combines a pastel-inspired interface with a curated set of tools for creators, developers, and everyday users. It removes the need for heavy post-install customization by delivering a polished, ready-to-use system from first boot.

---

## 🎯 Features

* 🎀 **Custom Pastel UI**

  * Ribbon-inspired theme (pink, blue, soft tones)
  * Clean desktop layout
  * Minimal, distraction-free design

* 🚀 **Preinstalled Applications**

  * Firefox (web browsing)
  * OBS Studio (recording/streaming)
  * Sublime Text & VSCodium (development)
  * Docker Engine (containerization)
  * Tilix (advanced terminal)

* 🎨 **Creative Tools**

  * GIMP (image editing)
  * Inkscape (vector graphics)
  * Audacity (audio editing)
  * Kdenlive / Shotcut (video editing)

* 🔧 **Utilities & System Tools**

  * VLC Media Player
  * KeePassXC (password manager)
  * Timeshift (system backups)
  * Stacer (system optimizer)
  * BleachBit (cleaner)

* 🌐 **Developer & API Tools**

  * Insomnia (API client)

* 📦 **Flatpak Support**

  * Flathub enabled by default
  * Flatseal preinstalled

* 🎀 **RibbonMail**

  * Custom-built email client (included)

---

## 🧠 Philosophy

Ribbon OS is built around three core ideas:

* **Aesthetic First** — A system that feels good to use
* **Ready Out of the Box** — No setup required
* **Lightweight but Capable** — Balance between performance and usability

## 💻 System Requirements

### Minimum Requirements

* **RAM:** 4 GB
* **Storage:** 25 GB available space
* **CPU:** 64-bit dual-core processor
* **Graphics:** Integrated graphics capable of 1024×768 resolution

### Recommended Requirements

* **RAM:** 8 GB or more
* **Storage:** 50 GB+ SSD
* **CPU:** 64-bit quad-core processor
* **Graphics:** GPU with 1080p support

---

Ribbon OS is designed to run smoothly on most modern systems while maintaining a lightweight and responsive experience.
